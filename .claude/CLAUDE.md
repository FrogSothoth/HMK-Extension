## Project Structure

This is a Fantasy Grounds extension for the HarnMaster Kethira (HMK) RPG system.

- `/docs/v0.01/` - Extension source code (XML, Lua, graphics)
- `/docs/extracted-sections/` - HarnMaster rulebook reference (markdown)
- `/docs/official-sources/` - Official HMK PDFs and spreadsheets

## Technology

- **Lua** - Game logic and scripting
- **XML** - UI layout and window definitions
- **Base Ruleset** - CoreRPG
